NPM Dependency Reader
====================
A stand-alone node script to find all the dependencies of a package

----------

Documents
-------------

**Run the script **

> node listPackages.js

**Output**

| Package Name  | Version |
| ------------- |:-------------|
| Component 1      | 1.0.2 |
| Component 2     | 3.2.5    |
| sub-comp 1 | 3.1.3      |
